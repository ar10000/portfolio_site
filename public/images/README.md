## Image Requirements

### Thumbnails
- **Size**: 600x400px (or 3:2 aspect ratio)
- **Format**: JPG or PNG
- **File size**: Keep under 200KB for fast loading
- **Optimization**: Use tools like TinyPNG or ImageOptim before uploading

### Screenshots
- **Size**: 800x600px (or 4:3 aspect ratio)
- **Format**: JPG or PNG
- **File size**: Keep under 300KB each
- **Optimization**: Compress before uploading

## How to Add Images

1. Take screenshots of your projects
2. Resize them to the recommended dimensions
3. Optimize them (compress file size)
4. Save them in the `public/images/projects/` folder with the exact filenames listed above
5. The code will automatically pick them up - no code changes needed!

## Quick Image Optimization Tools

- **Online**: [TinyPNG](https://tinypng.com/) or [Squoosh](https://squoosh.app/)
- **Mac**: ImageOptim (free app)
- **Command line**: `sips -Z 600 public/images/projects/your-image.jpg` (Mac)
